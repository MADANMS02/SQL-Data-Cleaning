This project demonstrates data cleaning techniques using MySQL on a real-world layoffs dataset.

- Data Cleaning
- UPDATE

- Standardized company names
- Handled NULL values
- Fixed date formats
- Cleaned inconsistent text values
- Prepared data for analysis
